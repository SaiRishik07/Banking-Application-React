# Banking Application

A modern full-stack banking application built using React, Spring Boot, PostgreSQL, JWT Authentication, and Docker.

## Features
- User Registration
- Secure Login
- Dashboard
- Money Transfer
- Deposit & Withdraw
- Transaction History
- Email Notifications
- Responsive UI

## Tech Stack
Frontend
- React
- CSS
- Axios

Backend
- Spring Boot
- Spring Security
- JWT
- PostgreSQL

Tools
- Docker
- Git
- GitHub
